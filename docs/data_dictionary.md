## 計算変数

| 変数名 | 計算式 | 説明 |
|--------|--------|------|
| `walkability_index` | `(zscore(population_density) + zscore(urban_ratio)) / 2` | 歩行環境複合指標 |
| `exercise_habit_rate` | `(Q3_rate + Q4_rate) / 2` | 運動習慣複合指標 |

## データ品質に関する注記

### 欠損データパターン
- **傾斜度データ:** 22/47都道府県が欠損（内陸部でDEMカバレッジなし）
- **欠損都道府県:** 青森、岩手、茨城、栃木、群馬、新潟、富山、石川、福井、山梨、長野、岐阜、静岡、愛知、三重、滋賀、京都、鳥取、島根、岡山、山口、徳島

### データ変換処理
1. **HbA1c:** カテゴリ中点の加重平均（例: "6.0-6.5" → 6.25）
2. **BMI肥満:** BMI≥25のカテゴリカウントから割合算出
3. **歩行環境:** population_densityとurban_ratioのZ-score標準化
4. **運動習慣:** Q3とQ4の回答率の平均

### バリデーションルール
- HbA1c: 5.5 ≤ 値 ≤ 7.0
- BMI肥満率: 10 ≤ 値 ≤ 50
- 歩行環境指数: -3 ≤ 値 ≤ +3（Z-score範囲）
- 高齢化率: 15 ≤ 値 ≤ 40
- 所得: 2000 ≤ 値 ≤ 5000（千円）
- 失業率: 1 ≤ 値 ≤ 5
